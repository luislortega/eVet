Descripcion de el proyecto:
-Este proyecto viene creado con el fin de crear un sistema de control animal para regular el abandono/perdida
y sacrifio de perros, con un sistema que estara conformado en tres etapas: 
  
  *Desarrollo de App para escritorio: (NetBeans - JAVA - MySql) 
         Con esta app se generaran los codigos QR que se imprimiran en la placa del perro, con este codigo 
         se generara el usuario y contraseña del dueño (para logearse en la plataforma) y poder recibir
         mensajes de quien encuentre a su perro mediante la APP.

               **NOTA: a esta aplicacion solo tendran acceso personas que sean verificadas por el gobierno 
               para utilizarlo, por lo que habria cierta restriccion en lo que vendria siendo el acceso. 
               a cada administrador de cada centro de control animal se le otorgara una cuenta y contraseña
               asi el podra crear los sub-usuarios que iran registrando a los animales en la base de datos,
               ellos podrian hacerlo en caso de que solo fuera uno. 

  *Desarollo Movil: (Android Studio - JAVA - SQLite)
         Con una app movil se leera el codigo QR y se buscara en una base de datos, nombre y foto de el perro, 
         numero de el dueño, (*) direccion del dueño (*) , su cartilla de vacunas y la posibilidad de contactar
         con el dueño desde la plataforma, se le podra enviar un mensaje desde la aplicacion para recibirlo en 
         la bandeja del usuario en la plataforma.

  *Desarollo web: (HTML5 - CSS - PHP - MySql)
         En la plataforma web habra un sistema donde se puedan logear, los dueños
